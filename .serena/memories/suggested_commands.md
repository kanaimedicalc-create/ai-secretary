# Suggested Commands

All commands run from `todo-next/`.

| Purpose | Command |
|---------|---------|
| Dev server | `npm run dev` |
| Production build | `npm run build` |
| Production start | `npm start` |
| Tests (one-shot) | `npm test` |
| Tests (watch) | `npm run test:watch` |
| Lint | `npm run lint` |
